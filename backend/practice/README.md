# 🥋 Carpeta de Práctica — Coding Dojo

Esta carpeta es tu **espacio de entrenamiento**. Aquí escribirás tus propias versiones de los archivos del proyecto para practicar.

## Cómo usar

1. Escribe `/dojo reescribe conexion.py` en el chat
2. Recibirás instrucciones de qué debe hacer el archivo
3. Crea tu versión aquí (ej: `mi_conexion.py`)
4. Pégalo en el chat para recibir feedback
