# рҹ§  MarketSense

**Plataforma de agentes inteligentes para anГЎlisis financiero**, potenciada por LLMs de AWS Bedrock y orquestada con LangGraph.

MarketSense no es un chatbot genГ©rico вҖ” es un **analista experto** capaz de consultar bases de datos SQL, buscar noticias financieras en tiempo real y ejecutar cГіdigo para generar reportes precisos del S&P 500.

## рҹ“Ӯ Estructura del Proyecto

```
в”ңв”Җв”Җ backend/          # CГіdigo Python principal
в”Ӯ   в”ңв”Җв”Җ agents/       # Agentes LangGraph
в”Ӯ   в”ңв”Җв”Җ tools/        # Herramientas: SQL, bГәsqueda, ejecuciГіn de cГіdigo
в”Ӯ   в”ңв”Җв”Җ etl/          # Pipelines de ingesta de datos (S&P 500)
в”Ӯ   в”ңв”Җв”Җ rag/          # GeneraciГіn Aumentada por RecuperaciГіn (RAG)
в”Ӯ   в”ңв”Җв”Җ db/           # Modelos SQLAlchemy y conexiГіn a RDS
в”Ӯ   в””в”Җв”Җ api/          # API FastAPI
в”ңв”Җв”Җ frontend/         # AplicaciГіn Next.js (panel de control)
в”ңв”Җв”Җ notebooks/        # Jupyter notebooks de exploraciГіn
в”ңв”Җв”Җ tests/            # Tests unitarios e integraciГіn
в””в”Җв”Җ infra/            # Infraestructura (Docker, IaC)
```

## рҹҡҖ Inicio RГЎpido

### Prerrequisitos

- Python 3.10+
- Node.js 18+
- Cuenta de AWS con acceso a Bedrock

### 1. Clonar y configurar entorno

```bash
git clone <url-del-repo>
cd IA-Generativa

# Configurar variables de entorno
cp .env.example .env
# Editar .env con tus credenciales
```

### 2. Instalar dependencias

```bash
# Backend (Python)
make instalar

# Frontend (Next.js)
make instalar-frontend
```

### 3. Desarrollo

```bash
# Backend
make dev-backend

# Frontend (en otra terminal)
make dev-frontend

# Jupyter notebooks
make notebook
```

### 4. Calidad de cГіdigo

```bash
make lint    # Linter + formateador (ruff)
make test    # Tests (pytest)
```
